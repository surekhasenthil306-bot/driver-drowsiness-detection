Driver Drowsiness Detection System

Project Description
This project is a real-time driver drowsiness detection system that helps prevent road accidents. It uses computer vision techniques to monitor the driver's eyes and alert them when signs of sleep or fatigue are detected.

Objective
To enhance road safety by detecting driver drowsiness and providing timely alerts.

Technologies Used
- Python
- OpenCV
- Dlib (optional)
- NumPy

How It Works
- Captures real-time video using a webcam
- Detects face and eyes
- Monitors eye closure (blink rate)
- If eyes are closed for too long, an alert is triggered
